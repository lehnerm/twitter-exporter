# Tweet Exporter
A simple command-line tool for exporting tweets from Twitter to a SQL database.

## Requirements

### Runtime
- Java 8 JDK installation
- Postgres compatible DB
- Twitter API credentials

### Development
- Scala SBT (https://www.scala-sbt.org/download.html)
